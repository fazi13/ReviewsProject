A) HTML Structure - top

This theme is a responsive layout based on Bootstrap. Here below an example of the homepage

If you would like to edit the color check the file css/colors.css you will find six pre-select color schemes. To add another color scheme add ID to the tag body as follow

<body id="yourcolor">

Explame:
<body id="green">

C) JavaScript - top

This theme imports five Javascript files.

  1. jquery.js
  2. bootstrap.min.js
  3. jquery.placeholder.js
  4. html5shiv.js
  5. custom.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Plugins can be included individually (though some have required dependencies), or all at once. Both bootstrap.js and bootstrap.min.jscontain all plugins in a single file.
  3. A jQuery plugin that enables HTML5 placeholder behavior for browsers that aren’t trying hard enough yet 
  4. Html5shiv code adds new HTML5 elements (which is simple code), but also supports printing HTML5 elements and includes the default styles for HTML5 elements, like block on article and section.
  5. All custom js code
